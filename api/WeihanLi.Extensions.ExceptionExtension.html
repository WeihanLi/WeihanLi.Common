IsFatal(Exception)

Determines whether the provided exception should be considered fatal. An exception is considered to be fatal if it, or any of the inner exceptions are one of the following type:

  • System.OutOfMemoryException
  • System.InsufficientMemoryException
  • System.ThreadAbortException
  • System.AccessViolationException
  • System.SEHException
  • System.StackOverflowException
  • System.TypeInitializationException
  • Microsoft.PowerApps.CoreFramework.MonitoredException marked as Fatal.
