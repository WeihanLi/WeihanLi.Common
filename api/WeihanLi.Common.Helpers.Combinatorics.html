Combinations<T>

Combinations defines a sequence of all possible subsets of a particular size from the set of values. Within the returned set, there is no prescribed order. This follows the mathematical concept of choose. For example, put 10 dominoes in a hat and pick 5. The number of possible combinations is defined as "10 choose 5", which is calculated as (10!) / ((10 - 5)! * 5!).

Permutations<T>

Permutations defines a sequence of all possible orderings of a set of values.

Variations<T>

Variations defines a sequence of all possible ordered subsets of a particular size from the set of values.
