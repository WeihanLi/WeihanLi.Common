Class PeriodicBatching<TEvent>

Base class for sinks that log events in batches. Batching is triggered asynchronously on a timer.

Remarks

To avoid unbounded memory growth, events are discarded after attempting to send a batch, regardless of whether the batch succeeded or not. Implementers that want to change this behavior need to either implement from scratch, or embed retry logic in the batch emitting functions.
